## Environment
- Anaconda3 5.1.0
- pycaffe

## Installation
- Step 1:
  Download the HCCR_caffemodel by running script in 'caffeModel/fetch_HCCR_caffemodel.sh'.
- Step 2:
  Install the caffe included pycaffe
- Step 3:
  Run 'python QTApplication.py' in the terminal.And then enjoy your journey.
